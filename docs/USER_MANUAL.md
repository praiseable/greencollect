## 🚀 Getting Started

### Creating an Account

1. **Visit the Web Portal** or **Download the Mobile App**
   - Web: https://gc.directconnect.services
   - Mobile: Available on Google Play Store and Apple App Store

2. **Register**
   - Click **"Sign Up"** or **"Register"**
   - Enter your details:
     - Full Name
     - Email Address
     - Phone Number (Pakistan format: 03XX-XXXXXXX)
     - Password (minimum 8 characters)
   - Accept Terms & Conditions
   - Click **"Create Account"**

3. **Verify Your Phone**
   - You'll receive an OTP via SMS
   - Enter the 6-digit code to verify your phone number

4. **Complete Your Profile**
   - Add a profile picture (optional)
   - Select your city/location
   - Choose your role: **Buyer**, **Seller**, or **Dealer**

### Logging In

- **Email/Phone + Password**: Enter your credentials and click "Login"
- **OTP Login**: Enter your phone number, receive OTP, enter code
- **Forgot Password**: Click "Forgot Password", enter email, check inbox for reset link

## 👥 For Buyers & Sellers

### Browsing Listings

#### Grid View
1. Go to **"Browse"** or **"Listings"** page
2. Use filters:
   - **Search**: Type keywords (e.g., "copper wire", "PET bottles")
   - **Category**: Select category (Metals, Plastics, Electronics, etc.)
   - **City**: Filter by location
   - **Price Range**: Set minimum and maximum price
3. Sort by:
   - Latest
   - Price: Low to High
   - Price: High to Low
   - Most Popular

#### Map View
1. Click the **"Map"** toggle button (top right)
2. See all listings as markers on the map
3. Click a marker to see:
   - Listing title
   - Price
   - Category
   - City
   - "View Details" link
4. Zoom in/out to explore different areas
5. Click **"Grid"** to switch back to card view

### Viewing Listing Details

1. Click on any listing card or map marker
2. View:
   - **Images**: Swipe through photos (up to 5)
   - **Price**: Displayed in PKR (₨)
   - **Quantity & Unit**: Available quantity
   - **Location**: City and pickup address
   - **Description**: Full details about the material
   - **Specifications**: Product attributes (purity, grade, etc.)
   - **Seller Info**: Name and contact details
   - **Map**: Interactive map showing exact pickup location
3. Actions:
   - **Contact Seller**: Send a message
   - **Call**: Direct phone call (if number available)
   - **Get Directions**: Open in Google Maps
   - **Share**: Share listing with others

### Creating a Listing

#### Step-by-Step Guide

1. **Click "Post Listing"** (top navigation or dashboard)

2. **Fill in Basic Information**
   - **Title** (required): Descriptive title (e.g., "500kg Copper Wire Scrap — Clean Grade A")
   - **Category** (required): Select main category
   - **Product Type** (optional): Select specific type if available
   - **Quantity** (required): Enter amount
   - **Unit** (required): Select unit (kg, ton, piece, etc.)
   - **Price** (required): Enter price in PKR (₨)
   - **Price Negotiable**: Toggle if price is negotiable
   - **Condition**: Select (New, Used, Refurbished, Scrap)

3. **Select Location**
   - **City** (required): Choose your city from dropdown
   - **Pickup Location** (required):
     - Click on the map to mark exact location
     - OR search for an address
     - OR click "My Location" to use GPS
   - **Address**: Auto-filled from map (editable)
   - **Contact Number**: Optional (defaults to your profile phone)

4. **Add Description**
   - Describe material quality, source, pickup details
   - Include any special instructions

5. **Upload Photos** (optional, max 5)
   - Click "Upload" button
   - Select images from your device
   - Remove photos by clicking the X icon

6. **Review and Submit**
   - Review all information
   - Click **"Post Listing"**
   - Wait for confirmation message

#### Tips for Great Listings
- ✅ Use clear, descriptive titles
- ✅ Upload high-quality photos
- ✅ Set accurate location on map
- ✅ Include detailed description
- ✅ Set competitive prices
- ✅ Mark price as negotiable if flexible

### Managing Your Listings

1. Go to **"Dashboard"** → **"My Listings"**
2. View all your posted listings
3. Actions:
   - **Edit**: Update details, photos, price
   - **Close**: Mark as sold/unavailable
   - **Delete**: Remove listing permanently
   - **View**: See listing as buyers see it

### Contacting Sellers

1. **Via Chat**
   - Click **"Contact Seller"** on listing page
   - Type your message
   - Send
   - Receive real-time notifications for replies

2. **Via Phone**
   - Click **"Call"** button (if seller provided number)
   - Direct call using your phone

3. **Via Map**
   - Click **"Get Directions"** to open in Google Maps
   - Navigate to pickup location

### Making Offers

1. View listing details
2. Click **"Make Offer"** (if enabled)
3. Enter your offer amount
4. Add optional message
5. Submit offer
6. Seller receives notification
7. Negotiate via chat if needed

### Notifications

- **Bell Icon** (top right): View all notifications
- Types:
  - New listings in your area
  - Messages from buyers/sellers
  - Offer updates
  - Transaction status
- Click notification to view details
- Mark as read individually or all at once

## 🔧 Troubleshooting

### Common Issues

#### Can't Login
- **Forgot Password**: Use "Forgot Password" link
- **OTP Not Received**: Check phone number, try again
- **Account Locked**: Contact support

#### Listing Not Appearing
- Check if listing is active
- Verify filters (category, city, price)
- Clear browser cache
- Try map view instead of grid

#### Map Not Loading
- **Web**: Check internet connection, allow location permissions
- **Mobile**: Enable GPS, allow location permissions
- Check if listing has valid coordinates

#### Image Upload Failed
- Check file size (max 5MB per image)
- Ensure image format (JPG, PNG)
- Check internet connection
- Try smaller image size

#### Payment Issues
- Verify payment gateway configuration
- Check account balance
- Contact support with transaction ID
