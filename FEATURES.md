# 📋 GreenCollect — Feature List

## ✅ MVP (Bare Minimum — Build First)

### 🏠 House Owner (Mobile App)
- [ ] Register / Login with phone number (OTP)
- [ ] Post garbage listing: photo, GPS location, address, garbage type, price expectation
- [ ] View status of posted listings (pending / collector assigned / collected / paid)
- [ ] Receive payment confirmation notification
- [ ] View earnings history

### 🚛 Local Collector (Mobile App)
- [ ] Register / Login (with verification)
- [ ] Receive push notifications for nearby garbage listings (radius-based)
- [ ] View map of nearby available listings
- [ ] Accept a garbage pickup job
- [ ] Navigate to house owner location
- [ ] Mark job as collected + confirm payment to house owner
- [ ] View personal collection inventory (by garbage type)
- [ ] Mark when collection point inventory is ready for regional sale

### 🏭 Regional Collector / Buyer (Mobile App + Web Portal)
- [ ] Register / Login
- [ ] View available bulk garbage by type and collection point
- [ ] Place purchase order for specific garbage types
- [ ] Confirm pickup from collection point
- [ ] View purchase history

### ⚙️ Admin (Web Portal)
- [ ] User management (approve/ban collectors)
- [ ] View all listings and statuses
- [ ] Basic analytics dashboard (total collected, by type, by region)
- [ ] Manage garbage types and pricing guidelines

### 🗃️ Garbage Types (MVP)
- Paper / Cardboard
- Plastic
- Metal / Iron Scrap
- Glass
- Raw / Organic Waste
- E-Waste
- Cloth / Textile
